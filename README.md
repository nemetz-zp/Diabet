### Diabet - примитивная база данных пациентов больных на сахарный диабет.
Основное назначение - формирование годовой заявки на закупку сахароснижающих препаратов.
##### Возможности приложения:
- установка фото пациента;
- ведение истории анализов пациента;
- ведение истории выдачи медикаментов пациенту;
- ведение дневника наблюдения пациента;
- создание базы данных используемых медикаментов (с указанием действующего вещества, дозировки, типа фасовки медикамента, цены и т.д.);
- быстрая замена назначений конкректного медикамента всем пациентам либо внутри територильного объединения;
- формирование Excel-отчётов по пациентам по разным критериям поиска (есть возможность защиты персональных данных пациента при формировании отчета);
